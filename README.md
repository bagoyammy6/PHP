# 工商邀約通訊錄
### 簡介
---
使用 XAMPP 以 PHP 搭配 MySQL 撰寫並具備 CRUD 功能
### 功能
---
* 使用者能新增邀約訊息
* 使用者能查看歷史訊息
* 使用者能編輯歷史訊息
* 使用者能刪除歷史訊息
* 歷史訊息頁面具有分頁查看功能
